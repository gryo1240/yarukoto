# きょうのやること

夫婦で共有できる、毎日のやることリスト(無料Webアプリ・PWA)。

**アプリ本体: https://gryo1240.github.io/yarukoto/**

📰 紹介記事: https://rinne-blog.com/yarukoto

毎日手書きしていた「今日のやること+何時にやるか」を自動化します。

- **定番タスクの自動展開**: 毎日/曜日ごと/週に○回/月に○回のルールで登録しておくと、毎朝自動でリストに並ぶ
- **時間の相談がしやすい**: 朝・昼・夕方・夜のブロック+決まっている予定だけ時刻指定。タップで簡単に変更
- **夫婦それぞれのスマホで同期**: 無料のFirebase Realtime Databaseを使ったリアルタイム同期(初期設定1回・手順はアプリ内の設定タブ)。設定しなければ1台ローカルでも使える
- **担当の色分け**: ニックネーム2人+「2人で」。「自分の分」フィルタつき
- **繰り越し**: 昨日終わらなかった予定は、翌朝「繰り越す/消す」を選べる
- **週に○回タスクの進み具合**: 「今週 1/3」のように完了ベースでカウント

データは自分たちのFirebase(または端末内)にのみ保存。第三者のサーバーには送信されません。

## 構成

| ファイル | 役割 |
|---|---|
| `index.html` | アプリ本体(HTML/CSS/JS 1ファイル。Firebase SDKは同期設定時のみCDNから読込) |
| `manifest.webmanifest` / `sw.js` | PWA化(ホーム画面インストール・オフライン対応) |
| `icon-*.png` | アプリアイコン |

`main` に push すると GitHub Pages に自動デプロイされます。
